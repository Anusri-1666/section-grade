# section/grade management system
 
This project is a simple C based student grade management system.
it allows users to update,delete,view and store student records.

---

## Project Title
Section/grade mnanagement system 

---

## Functional requirements
### 1.Add student 
    -enter roll number,name and marks.
    -store details in file('grade.dat').

### 2. View students
   -Display all the saved student records.

### 3. Search Student 
   -Find a student using roll number.

### 4. Update Student
   -Modify marks or name of an existing student.

### 5. Delete Student
   -Remove a student record from the file.

### 6. Save and exit
   -Save all changes permanently to the file.

---

## Features
 -File handling using '.dat' file.
 -Menu-driven program.
 -Handles multiple student records.
 -Prevents duplicate roll numbers.
 -User-friendly test interface.
 -Easy to compile and run.
